# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [26.1.2.1] - 2026-04-15

### Added
- Dash charge bar on the HUD when riding a Happy Ghast (replaces XP bar, like the horse jump bar)
- Configurable keybind (default: G) for charging the dash
- Proportional cooldown — shorter charges mean shorter cooldowns
- Overcharge decay — holding past max charge gradually reduces power
- Vertical boost during dash to maintain altitude
- Fabric and NeoForge support (Minecraft 26.1.2)
- In-game test functions (`ghastrider:test_ride`, `ghastrider:test_setup`)
- English and French translations
