# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| Latest release | Yes |
| Older releases | No |

## Reporting a vulnerability

If you discover a security vulnerability in GhastRider, please report it responsibly:

1. **Do not** open a public issue
2. Email **quatrei.gaspard@gmail.com** with a description of the vulnerability
3. Include steps to reproduce if possible
4. You will receive a response within 7 days

Thank you for helping keep GhastRider safe.
